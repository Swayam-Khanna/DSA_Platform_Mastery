🚀 DSA Mastery – AI Powered Data Structures & Algorithms Platform

An AI-powered DSA learning platform designed to help developers master Data Structures & Algorithms and crack FAANG interviews.

It provides curated problems, structured learning roadmaps, AI tutoring, and competitive leaderboards to track progress and improve problem-solving skills.

⚡ Getting Started
1️⃣ Clone the Repository
git clone https://github.com/your-username/dsa-mastery.git
cd dsa-mastery
2️⃣ Install Dependencies

Frontend

cd frontend
npm install

Backend

cd backend
npm install
3️⃣ Environment Variables

Create .env file in backend:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
OPENAI_API_KEY=your_api_key
4️⃣ Run the Project

Start backend:

npm run dev

Start frontend:

npm start
